# templates

- `ai-article-summarizer/`: Next.js 16 reading workspace template with AI article summaries, summary history, and Markdown-to-HTML export.
- `json-yaml-xml-parser/`: TypeScript library + CLI that parses JSON, YAML, and XML into a normalized, JSON-serializable value (TypeScript 6, Node.js ESM, Vitest 4).
- `nodejs-cli-file-search/`: TypeScript Node.js CLI that recursively searches files by name, skipping ignored and unreadable directories (TypeScript 6, Vitest 4).
- `random-red-black-tree-sort/`: TypeScript red-black tree that sorts 10,000 randomly inserted elements via in-order traversal, with full invariant validation (TypeScript 6, Vitest 4).
- `trumpet-twitter/`: Full-stack "micro Twitter" template — React 19 + Vite 8 front end, Express 5 API, SQLite (better-sqlite3), shared Zod 4 contracts, pnpm workspace.
- `web-front-end-snake-game/`: Pure front-end TypeScript Snake game with a fixed-timestep loop, Canvas rendering, and persisted best score (TypeScript 6, Vite 8).
